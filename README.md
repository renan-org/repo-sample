## Workflows

This repository contains several GitHub Actions workflows for testing and experimentation:

| Workflow | Description |
|----------|-------------|
| **CodeQL Advanced** | Automated code scanning for security vulnerabilities using CodeQL analysis |
| **Admin Team Management** | Automates admin team membership requests via issue-based workflows |
| **Test Admin Team Management** | Manual workflow dispatch for testing admin team operations |
| **Security Scan** | Reusable security scanning workflow integration from `shared-security-scan` |
| **Merge Gatekeeper** | Ensures all required checks pass before allowing merges |
| **ARC Workflow** | Tests self-hosted runners with Actions Runner Controller (ARC) |
| **Label Workflow** | Handles PR label events (e.g., `review-completed` label logic) |
| **Dummy Reusable Workflow** | Demonstrates reusable workflow patterns with local custom actions |
| **Test Reusable Locally** | Tests reusable workflows locally before releasing/tagging |
